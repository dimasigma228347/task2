моя библиотека предлагает пользователю добавить книгу, редактировать уже существующие, вывести всю библиотеку, вывести прочитанные или избранные книги. библиотека сохраняется после каждого действия, также можно убрать пометку прочитанная ил избранная при повторном обозначении. библиотека сохраняется в файл data.json при его отсутствии создает его. также библиотека загружается автоматически при запуске программы.
